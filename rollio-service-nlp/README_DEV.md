# Rolio Python Dev Environment Info

## Conda Environment
1. Activate correct conda env
```
conda env list
conda activate *env*
```

2. Ensure conda environment has correct packages
```conda install pip spacy re numpy pathlib keras sklearn en_core_web_lg```
